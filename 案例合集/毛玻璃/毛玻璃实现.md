实现起来很简单,`stack`叠加就行了`BackdropFilter`
```
BackdropFilter(
  filter: ImageFilter.blur(sigmaX: 10, sigmaY: 10),  
  child: Container(  
    width: 360.w,  
    height: 690.h,  
  ),  
),
```

```
Stack(  
  children: [  
    GestureDetector(  
      child: Container(  
        decoration: BoxDecoration(color: Colors.white),  
        width: 360.w,  
        height: 690.h,  
        child: Image.file(  
          File('这里是你的图片'),  
          fit: BoxFit.contain,

    ),  
 BackdropFilter(

 ],
```
